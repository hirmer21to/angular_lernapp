My first Angular-App:

Let's start

First open your command line and type:

$ ng new my-first-app

A project with all files and folders will be created.

These are:

  • The new workspace, with the root folder "my-first-app".
  • An initial skeleton app project, also called "my-first-app" that you will find it in the subfolder "src".
  • An end-to-end test project in the subfolder "e2e".
  • Related configuration files.

$ cd my-first-app

Then you can navigate to that project-folder.

$ ng serve --open

With "serve" you can already run the application and "--open" opens the browser directly on http://localhost:4200/ and shows your brand new application.
