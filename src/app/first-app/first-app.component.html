My first Angular-App:

Let's start

First open your command line and type: 

ng new my-first-app

You will get the question "? Would you like to add Angular routing?". Answer it with "Yes"! In the next step you can choose which stylesheet format you would like to use. In this tutorial we use CSS.

A project which includes all files and folders will be created.

These are:

  • The new workspace with the root folder "my-first-app".
  • An initial skeleton app project, also called "my-first-app" which you will find it in the subfolder "src".
  • An end-to-end test project in the subfolder "e2e".
  • Related configuration files.
cd my-first-app

Then you can navigate to that project-folder. If you get an answer like this from your system: added 1098 packages from 1166 contributors and audited 39144 packages in 35.242s found 3 vulnerabilities (2 low, 1 high), just follow the instructions on your command line (npm audit fix or similar) and it will be fine.

ng serve --open

With serve you can already run the application and --open opens the browser directly on http://localhost:4200/ and shows your brand new application.
