Project Introduction and Overview

What am I able to do after this Tutorial?

Probably the most interesting question: What will I be able to program after working through this project?

  • Create an application with many reusable components.
  • Show a list of items by using built-in Angular directives.
  • Use directives to show and hide items.
  • Create, read, update and delete items of a list.
  • Create forms.
  • Use one-way and two-way binding.
  • Bind methods to user events like button click.
  • Use routing to navigate through components and their views.
  • Format data by using pipes.
  • Use a service to manage the whole functionality.

What is this project about?

Every student might know the problem. It's a new semester, you have a lot of lectures and you never know which room to go next. Wouldn't it be nice to have an app shows you where to go at first sight by just opening it.

User Story

As a student
I want to have an app that shows me the room of my very next lecture,
to see where to go next at first sight.

So this is basically the whole idea. On the first page we will display the next lecture of the day. We will also have lecture data to manage. It will be possible to view all lecutres in a timetable, to create new or update existing lectures, or delete them.

Wireframes

When you start the app you will see the next lecture you have to go to on the "Home-Screen". Above the Home-Screen you can find the navigation.

When you click on Timetable you will get to the list of lectures you have. Here you can see that every lecture can be deleted. You can also delete all lectures with one click. Furthermore you can click the "Add"-button to add a new lecture. With a click on one lecture you will get to the details of this specific lecture and you can change them.

So here you can change some details and save them. Then you go back to the Timetable and see the updated list of lectures.

The last point in the menu is "Add". From there, or with the "Add"-button on the timetable-view you can get to the add-view. There you can add a totally new lecture by making some input and choosing options from the drop-downs.
