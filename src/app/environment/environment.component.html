How to set up the environment:

The Editor

As a perfect code editor for Angular I would recommend Visual Studio Code. It's for all operating systems and comes with an own Git integration and support for TypesScript.

If you don't have it already, just download it:

Visual Studio Code Download Link

Node.js

Node.js is a runtime environment for running JavaScript on the server. It can be used for developing frondend and backend. It comes with the Node Package Manager (NPM).

If you don't have Node, you can download and install it:

Node.js Download Link

Next you should try to get the version number on your command line:

$ node -v

$ npm -v

Angular requires Node.js version 8.x or 10.x, you always need to keep it up to date.

Now you can install packages like this:

$ npm install your-package-name

Or global like this:

$ npm install -g your-package-name

Angular-CLI

Angular-CLI is a commandline tool based on Node.js. It helps you from the beginning. When you start a new project, the CLI prepares everything for you: Creates your files, installs the npm-packages and creates Git for you.

It automatically generates skeletons for components, services, pipes, and directives. And you can get it like this:

$ npm install -g windows-build-tools

$ npm install -g @angular/cli
